//HOW TO USE//

Pixel chess contains 2 type of pieces the 16x32p pack and the 16x16p and two colors for each one. A board and a pointer (mouse).

16x16p: Contains a sprite sheet for each color
16x32p: Contains a sprite sheet for each color + the individual sprite for each piece W(white) and B(black)
Boards: Contains 2 board types, one top-down and other with a little perspective (142px142p) w/ 3 color variations

## Assets
This project uses third-party pixel art assets.

Download the asset pack from:
https://dani-maccari.itch.io/pixel-chess

From the ZIP file, extract:
- `16x16 pieces` PNG (path: `16x16 pieces`)
- `142x142 board` PNG (path: `boards`, with 12px padding)

Place the extracted files into: resources/checknull/

Assets are not included in this repository due to license restrictions.
